# JACARE DO PAPO AMARELO INFO APP

## (React, Tailwindcss, Vite)

### Comandos para rodar o projeto:
```


baixar as dependencias:
- dentro da pasta \app rode o comando > npm install 

rodar o projeto localmente:
npm run dev

caso o tailwindcss de algum erro: npm i tailwindcss@3 postcss autoprefixer

npx tailwindcss -i ./src/input.css -o ./static/css/output.css --watch (corrigir)
npx tailwindcss -i ./src/input.css -o ./static/css/output.css --minify (corrigir)

```
